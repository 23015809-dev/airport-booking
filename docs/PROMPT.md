# PROMPT.md — Airport Transfer Booking System

## Task
Build and maintain a complete Laravel web application for airport/tourist transfer booking.
Implement only the required features.

## Stack
- Backend: PHP 8.2+ / Laravel
- Frontend: Blade + plain HTML/CSS/JS
- Database: MySQL
- Image upload: Laravel public storage

## UI Rules
- Consistent design tokens and reusable components
- Accessible focus and hover states
- Clear text/background contrast
- English-only UI text

## Development Order
1. App setup and DB config
2. Layouts and base styles
3. Migrations and models
4. Auth and role authorization
5. Admin CRUD modules
6. User booking flow
7. Validation and upload handling
8. Final QA and responsive checks

## Constraints
- No Tailwind, Bootstrap, Alpine, Vue, React
- No npm/Node tooling required for app runtime

## Change Log (2026-05-22)
- UI language switched to English for all runtime screens.
- No npm/Node runtime steps are required.
- Added and applied global CSS override file (public/css/site.css).
- Per-screen UX polish completed for user and admin modules.
